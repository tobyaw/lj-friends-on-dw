h1. Dreamwidth tools

Some handy utilities to use with Dreamwidth.

h2. lj_friends_on_dw.rb

This is a simple Ruby script that takes one or more LJ usernames on the command line, and prints out a list of which of their LJ friends exist as accounts on Dreamwidth.

h2. lj-friends-on-dw (folder)

This is an HTML page with JavaScript and PHP code that does a similar task.
